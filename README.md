# CS2006-Calculator-H1

## running the program
```
ghc --make Main.hs -o calc 
./calc
```

### running the calculator 
```
ghci Main.hs 
main
```

### reading commands from a file 
```
ghci Main.hs 
runFile "filename.txt"
```

## calculator commands 
- `:q` = quit
- `:1` = go back to command 1 
- `:c` = clear command history
- `#`  = comment
